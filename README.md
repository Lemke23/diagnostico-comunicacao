# Diagnóstico de Comunicação

Projeto simples com HTML, CSS e JavaScript puro para coleta de informações iniciais sobre comunicação comercial.

## Funcionalidades

- Formulário com nome e empresa
- Perguntas sobre Instagram, atendimento e vendas
- Campo opcional para detalhes adicionais
- Validação de campos obrigatórios
- Mensagem de sucesso após envio
- Layout responsivo

## Arquivos

- `index.html`
- `style.css`
- `script.js`
- `agents.md`

## Uso

Abra `index.html` em um navegador.
