# Agents

## Objetivo

Este projeto contém um formulário simples de diagnóstico de comunicação com foco em Instagram, atendimento e vendas.

## Arquivos

- `index.html`: estrutura da página e do formulário
- `style.css`: aparência, responsividade e estados visuais
- `script.js`: validação e mensagem de sucesso

## Comportamento esperado

- Campos obrigatórios: nome, empresa, Instagram, atendimento e vendas
- Campo opcional: descrição do desafio principal
- Validação no envio e durante interação com os campos
- Exibição de mensagem de sucesso após envio válido

## Stack

- HTML
- CSS
- JavaScript
